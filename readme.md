## github branch 설계 규칙
### Branch 확인 하기
### backend와 frontend로 banch를 나눠놨으니 본인이 해당하는 브랜치에 들어가 본인의 브랜치 생성하기 

1. 현재 내가 위치한 Branch 확인
   - `$git branch`
2. 원격 저장소의 브랜치 확인
   - `$git branch -r`
3. 브랜치의 마지막 커밋 메세지 확인
   - `$git branch -v`

### Branch 생성 및 이동
1. Branch 생성하기
    - git branch 브랜치명
        - `$git branch test`
2. 생성한 Branch로 이동하기
    - git checkout 브랜치명
        - `$git checkout test`

### Branch 삭제
git branch -d 브랜치명  
`$git branch -d test`

### branch 병합 Git Merge
`$git merge 브랜치명`

### master branch로 이동
`$git checkout master`
